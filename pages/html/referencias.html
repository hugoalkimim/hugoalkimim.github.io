Tag Descrição
<p> Define um parágrafo.
<title> Define um título para o documento.
<area> Define a area dentro de um image map.
<meta> Define metadata sobre um documento HTML.
<tbody> Agrupa o body content em uma tabela.
<dd> Define a descrição/valor de um termo em uma lista de descrições.
<col> Útil para aplicar estilos para uma coluna inteira, ao invés de repetir os estilos para cada célula, para cada linha.
<base> Especifica uma URL e um alvo padrão para todos os links em uma página.
<script> Define um script(JavaScript) do lado do cliente.
<th> Define uma célula de cabeçalho em uma tabela HTML.
<colgroup> Especifica um grupo de uma ou mais colunas em uma tabela para formatação.
<ul> Define uma lista não ordenada (com marcadores).
<bdi> Isola uma parte do texto que pode estar formatada em uma direção diferente de outro texto fora dele.
<main> Especifica o conteúdo principal de um documento.
<figure> Especifica conteúdo independente.
<td> Define uma célula em uma tabela.
<dl> Define uma lista de descrição.
<section> Define uma seção em um documento.
