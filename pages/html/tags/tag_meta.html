Definição e Uso

A tag <meta> define metadados sobre um documento HTML. Metadados são dados (informações) sobre dados.
<meta> tags sempre vão dentro do elemento <head> e são normalmente usadas para especificar conjunto de caracteres, descrição de página, palavras-chave, autor do documento e configurações de janela de visualização.
Os metadados não serão exibidos na página, mas podem ser analisados ​​pela máquina.
Os metadados são usados ​​por navegadores (como exibir conteúdo ou recarregar a página), mecanismos de pesquisa (palavras-chave) e outros serviços da web.
Existe um método para permitir que os web designers controlem a janela de visualização (a área visível do usuário em uma página da web), por meio da tag <meta> (consulte o exemplo "Configurando a janela de visualização" abaixo).

Exemplo

Descreva os metadados em um documento HTML:

            
                <head>
                    <meta charset = "UTF-8">
                    <meta name = "descrição" content = "Tutoriais da Web grátis">
                    <meta name = "palavras-chave" content = "HTML, CSS, JavaScript">
                    <meta name = "autor" content = "John Doe">
                    <meta name = "viewport" content = "width = device-width, initial-scale = 1.0">
                </head>
