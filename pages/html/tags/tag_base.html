Tag <base>

Referencias Home Sobre

Definição e Uso

A tag <base> especifica a URL/alvo base para todas relativas URL's em um documento.
Poderá existir no máximo uma tag <base> em um documento, e deve estar dentro da tag <head>.

Exemplo

Especifica uma URL e um alvo padrão para todos os links em uma página: 

            
                <head>
                    <base href="https://www.w3schools.com/" target="_blank">
                </head>

                <body>
                    <img src="images/stickman.gif" width="24" height="39" alt="Stickman">
                    <a href="tags/tag_base.asp">HTML base Tag</a>
                </body>

Atributos

Atributos Valor Descrição
href URL Especifica a URL base para todas as URL's relativas da página
target _blank
_parent
_self
_top
framename 		Especifica o alvo padrão para todos os hyperlinks e formulários da página
