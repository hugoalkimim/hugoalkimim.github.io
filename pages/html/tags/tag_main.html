Tag <main>

Definição e Uso

A tag <main> especifica o conteúdo principal de um documento.
O conteúdo dentro do <main> elemento deve ser exclusivo para o documento. Não deve conter nenhum conteúdo que se repita em documentos, como barras laterais, links de navegação, informações de direitos autorais, logotipos de sites e formulários de pesquisa.
Nota: Não deve haver mais de um <main> elemento em um documento. O <main> elemento NÃO deve ser descendente de um elemento <article>, <aside>, <footer>, <header> ou <nav>.

Exemplo

Especifique o conteúdo principal do documento:

            
                <main>
                    <h1>Navegadores mais populares</h1>
                    <p>Chrome, Firefox e Edge são os navegadores mais usados ​​atualmente.</p>
                  
                    <article>
                        <h2>Google Chrome</h2>
                        <p>Google Chrome é um navegador desenvolvido pelo Google, lançado em 2008. Chrome é o navegador mais popular do mundo hoje!</p>
                    </article>
                  
                    <article>
                        <h2>Mozilla Firefox</h2>
                        <p>Mozilla Firefox é um navegador de código aberto desenvolvido pela Mozilla. O Firefox é o segundo navegador mais popular desde janeiro de 2018.</p>
                    </article>
                  
                    <article>
                        <h2>Microsoft Edge</h2>
                        <p>Microsoft Edge é um navegador desenvolvido pela Microsoft, lançado em 2015. O Microsoft Edge substituiu o Internet Explorer.</p>
                    </article>
                </main>

Atributos

A tag <main> também suporta os atributos globais em HTML.
A tag <main> também suporta os atributos de evento em HTML.
