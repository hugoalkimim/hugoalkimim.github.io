Tag <dd>

Referencias Home Sobre

Definição e Uso

A tag <dd> é usada para descrever um termo / nome em uma lista de descrição.
A tag <dd> é usada em conjunto com <dl> (define uma lista de descrição) e <dt> (define termos / nomes).
Dentro de uma <dd>tag você pode colocar parágrafos, quebras de linha, imagens, links, listas, etc.

Exemplo

Uma lista de descrições, com termos e descrições:

            
                <dl>
                    <dt>Café</dt>
                    <dd>Bebida quente preta</dd>
                    <dt>Leite</dt>
                    <dd>Bebida fria branca</dd>
                </dl>

Atributos

A tag <dd> suporta os Atributos Globais e os Atributos de Eventos.
