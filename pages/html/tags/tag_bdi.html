Tag <bdi>

Referencias Home Sobre

Definição e Uso

BDI significa isolamento bidirecional.
A tag <bdi> isola uma parte do texto que pode ser formatada em uma direção diferente de outro texto fora dela.
Este elemento é útil ao incorporar conteúdo gerado pelo usuário com uma direção de texto desconhecida.

Exemplo

Isole os nomes de usuário das configurações de direção do texto ao redor:

            
                <ul>
                    <li>User <bdi>hrefs</bdi>: 60 points</li>
                    <li>User <bdi>jdoe</bdi>: 80 points</li>
                    <li>User <bdi>إيان</bdi>: 90 points</li>
                </ul>

Atributos

A tag <bdi> também suporta os atributos globais em HTML.
A tag <bdi> também suporta os atributos de evento em HTML.
