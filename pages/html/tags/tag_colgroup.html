Tag <colgroup>

Referencias Home Sobre

Definição e Uso

A tag <colgroup> especifica um grupo de uma ou mais colunas em uma tabela para formatação.
A tag <colgroup> é útil para aplicar estilos a uma coluna inteira, ao invés de repetir os estilos para cada célula, para cada linha.
Nota: A tag <colgroup> precisa ser um filho de uma tag <table>, depois da tag <caption> e antes de quaisquer tags <thead>, <tbody>, <tfoot>, e <tr>.
Dica: Para definir diferentes propriedades para uma coluna com a tag <colgroup>, use a tag <col> con a tag <colgroup>.

Exemplo

Define a cor de fundo de três colunas com a tags <colgroup> e <col>: 

              
                  <table>
                      <colgroup>
                          <col span="2" style="background-color:red">
                          <col style="background-color:yellow">
                      </colgroup>
                      <tr>
                          <th>ISBN</th>
                          <th>Title</th>
                          <th>Price</th>
                      </tr>
                      <tr>
                          <td>3476896</td>
                          <td>My first HTML</td>
                          <td>$53</td>
                      </tr>
                </table>

Atributos

Atributo Valor Descrição
span number Especifica o número de colunas que um grupo de colunas que a deve atuar
