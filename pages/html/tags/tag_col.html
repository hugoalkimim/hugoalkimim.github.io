Definição e Uso

A tag <col> especifica propriedades para cada coluna com a tag <colgroup>.
A tag col <col> é útil para aplicar estilos para uma coluna inteira , ao invés de repetir os estilos para cada célula, para cada linha.

Exemplo

Define a cor de fundo de três colunas com as tags <colgroup> e <col>. 

            
                <table>
                    <colgroup>
                        <col span="2" style="background-color:red">
                        <col style="background-color:yellow">
                    </colgroup>
                    <tr>
                        <th>ISBN</th>
                        <th>Title</th>
                        <th>Price</th>
                    </tr>
                    <tr>
                        <td>3476896</td>
                        <td>My first HTML</td>
                        <td>$53</td>
                    </tr>
                </table>

Atributos

Atributo Valor Descrição
span number Especifica o número de colunas que a tag <col> deve atuar
