Definição e Uso

A tag <p> define um parágrafo.
Os navegadores adicionam automaticamente uma única linha em branco antes e depois de cada <p> elemento.
Dica: Use CSS para estilizar parágrafos.

Exemplo

Um parágrafo é marcado da seguinte forma:

            
                <p> Este é um texto em um parágrafo. <p>
